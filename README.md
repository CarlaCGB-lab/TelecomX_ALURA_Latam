# Desafío TelecomX_ALURA_Latam

## Introducción

Este proyecto tiene por objetivo analizar la evasión de clientes (Churn) en la empresa Telecom-X Churn o tasa de abandono, es un término muy usado en análisis de datos y negocios que se refiere a cuando los clientes dejan de usar un servicio o dejan de ser clientes de una empresa.

Mediante el analisis de la base de datos se pretende identificar las variables que influyen en dicho término, para poder generar estrategias y reducir el churn.

## Estructura del proyecto

- `READMED.md`: Descripción del proyecto y su contenido.
- `TelecomX_Data.json`: Base de datos.
- `TelecomX_LATAM.ipynb`: Código en python..
